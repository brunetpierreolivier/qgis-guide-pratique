# Checklist — Avant livraison

## Fichiers
- [ ] Un seul format principal : GeoPackage `.gpkg`
- [ ] Nommage clair des fichiers (pas de “final_final2.gpkg”)
- [ ] Dossier `livraison/` propre (pas de cache, pas de doublons)

## QGIS
- [ ] Projet `.qgz` sauvegardé et s’ouvre sans erreurs
- [ ] Styles OK (si nécessaire) : couleurs lisibles, étiquettes propres
- [ ] Métadonnées minimales (source, date, CRS)

## Contrôle rapide
- [ ] Affichage correct (localisation, échelle)
- [ ] Mesure surface/distance cohérente
- [ ] Vérif de validité géométrique faite

## Packaging
- [ ] Un README livraison (2–5 lignes : contenu + CRS + source)
- [ ] Si plusieurs couches : liste des couches + rôle de chacune
